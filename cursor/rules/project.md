Este é um projeto em Next.js (App Router).

Stack:
- Next.js
- Supabase
- Tailwind CSS

Regras:
- Sempre usar componentes reutilizáveis
- Manter padrão modular já existente
- Não quebrar estrutura de pastas
- Código limpo e tipado
- Evitar duplicação

Contexto do sistema:
- Plataforma de busca de espaços para eventos
- Existem dois perfis: proprietário e cliente final
- Dashboard com métricas e gestão de propriedades

Ao sugerir código:
- Sempre explique rapidamente
- Depois entregue o código pronto
